# Hadris CLI

General-purpose command-line utility for filesystem and disk operations.

> **Note:** This crate is a work in progress. For specialized functionality, see:
> - [hadris-fat-cli](../hadris-fat-cli) - FAT filesystem operations
> - [hadris-iso-cli](../hadris-iso-cli) - ISO 9660 operations

## Installation

```bash
cargo install hadris-cli
```

## Planned Features

- Unified interface for multiple filesystem types
- Partition table operations (MBR/GPT)
- Disk image manipulation
- Format conversion
