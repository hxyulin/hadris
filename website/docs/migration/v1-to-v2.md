# Migrate from Hadris 1.x

The full migration guide is maintained in the repository:

- [Hadris 1.x to 2.0 migration guide](https://github.com/hxyulin/hadris/blob/main/docs/hadris-1-to-2-migration.md)

The most important changes are explicit sync/async namespaces, independent
feature dimensions, normalized lifecycle verbs, and canonical type names such
as `FatVolume` and `PartitionTable`.
